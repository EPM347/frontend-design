01 · Soul

The brand in one paragraph.

When in doubt, read this. It anchors everything below.

OneClick·Art is the friendliest art tool on the internet. It's warm cream paper, thick brown ink, a small chunky stamp on the title — you'd find it in a thoughtful stationery shop, not a tech expo.

The voice is direct and unhurried. Hemingway, not Shakespeare. "Tap a style. Get art." — short sentences with periods that land. The product does one thing well; the design says so without raising its voice.

The neo-brutalist accents (chunky borders, hard offset shadows, slightly rotated stamps) keep the warmth from feeling sleepy. They tell visitors: we know what we're doing, we're confident enough to be playful. They never dominate — they punctuate.

One rule above all others: if a design choice would feel out of place in a small printmaker's shop run by a person who genuinely loves what they make, it doesn't belong in OneClick·Art.

Usage rules

Cream

Cream is the canvas.

Pages start on cream-50 or white. Cream-100 is for inset surfaces (sidebars, sticky strips). Avoid full-page cream-200 — it reads as washed-out paper.

Cocoa

Cocoa for everything that draws the eye.

All borders are cocoa-900. All body text is cocoa-900. All hard shadows are cocoa-900. The cocoa palette gives the brand its warmth — never substitute black.

Terracotta

Terracotta is rare.

Marks the thing on a screen the user should do, or the one word in a headline that carries the meaning. If everything is highlighted, nothing is.

Butter

Butter is the friendly accent.

Pill eyebrows, boxed numbers, reassurance callouts. Always paired with a cocoa border + small offset shadow. Never used as a CTA color — that's terracotta's job.

Sage

Sage is the tick mark.

Feature lists use sage for the ✓. Status pills use it for "Ready". Never use sage for a CTA — it reads as "this isn't the action."

Tomato

Tomato is forbidden in marketing.

It belongs only in: error states, danger-zone account actions, failed-job indicators. Never in pricing, never in CTAs, never in headlines.

04 · Neo Primitives

Five small visual moves.

The neo-brutalist accents that punctuate the warmth. Use sparingly — they're punctuation, not paragraphs.

1. Chunky border + hard shadow

Always 2.5–4px border, never 1px. Always hard offset shadow, never blurred. The default for any "card" surface. Match the shadow size to the focal weight.

border: 3px solid var(--cocoa-900); box-shadow: 6px 6px 0 0 var(--cocoa-900); /* never with blur or spread */
A card surface.

2. Press-down active state

Buttons get a physical press: shadow goes to 0 0, the element shifts by exactly the shadow's offset. Without this, buttons feel dead.

.btn:active { transform: translate(4px, 4px); box-shadow: 0 0 0 0 var(--cocoa-900); }
CLICK ME

3. Boxed word — terracotta + rotation

The signature treatment. Wraps one word in a rotating, terracotta-stamped box. Rule of one — never two boxed words in the same composition.

background: var(--terracotta-500); border: 4px solid var(--cocoa-900); box-shadow: 6px 6px 0 0 var(--cocoa-900); transform: rotate(-2deg); padding: 0.04em 0.18em;
Make your first piece.

4. Boxed number — butter + rotation

Same treatment, butter background. For numbers that are the protagonist: counts, prices, days-in-a-row. Same rule of one.

background: var(--butter-500); border: 3px solid var(--cocoa-900); box-shadow: 4px 4px 0 0 var(--cocoa-900); transform: rotate(-2deg);

5. Pill eyebrow with shadow

The above-headline label. Compresses a value prop into 4–6 words. Always butter background, terracotta text, cocoa border + offset shadow.

07 · Imagery

Real photos beat AI fakes.

OneClick·Art is built around real human photos. Marketing should feature real before-after pairs with rights-cleared subjects. Square crops dominate.

Yes

Warm, human, slightly imperfect.

A real face. A pet on a couch. A wedding shot. Subtle film grain or paper texture in backgrounds is welcome — feels like a printmaker's shop.

No

Glossy stock photography.

The woman pointing at a laptop. The diverse-handshake-over-conference-table shot. Too corporate.

No

Generic "AI art" tropes.

Glowing particles, neon orbs, abstract light rays, circuit-board "digital" textures. The product is photo→art, not abstract generation. These tropes undermine the whole pitch.

No

Tech imagery.

Robot hands. Hands on glowing keyboards. Brain-with-circuits. The product hides AI; don't undo that with imagery.

08 · Don'ts

Things that break the brand instantly.

If you find any of these in production, fix them immediately. Each undermines a specific brand value.

Don't · 01

Add a fifth color.

Cream + cocoa + terracotta + butter + sage. That's the system. Want a new mood? Lean on existing colors at different weights.

Don't · 02

Use Inter, Roboto, or generic sans.

They're crisp but cold. DM Sans has personality. Don't add a fourth typeface; three is enough.

Don't · 03

Blur the shadows.

Hard offset only. box-shadow: 4px 4px 0 0 — note the 0 0. No blur, no spread.

Don't · 04

Say "AI" in marketing headlines.

The product hides AI. Headlines say what visitors get, not what's under the hood. (SEO surfaces excepted.)

Don't · 05

Use generic loading copy.

"Cooking..." beats "Loading..." Every time. The voice extends to the moments between actions.

Don't · 06

Two boxed-words in one composition.

Rule of one. The treatment loses meaning if it's used decoratively. One stamp per page.

Don't · 07

Market features users can't see.

"Powered by SDXL 1.0 + LoRA fine-tunes" is not for the homepage. Show what they make, not how.

Don't · 08

Use pure black or gray.

Substitute cocoa-900 (#3A2E26) for black, cocoa-700 (#5C4A3D) for gray. The cocoa palette is what gives the brand its warmth.

Don't · 09

Break the press-down state.

It's the difference between feeling alive and feeling dead. Every CTA needs the active translate + shadow-to-zero.

Don't · 10

Optimize for designers.

Optimize for someone who just wants their pet's portrait. If a beautiful design choice would confuse that person, the choice is wrong.

09 · Tokens

Every CSS variable, in one place.

All tokens live in app/app.css under .theme-cream. Wrap any subtree in <div className="theme-cream"> and every Tailwind semantic token flips to the cream palette automatically. 

/* All tokens scoped to .theme-cream — wrap any subtree to flip */
.theme-cream {
  /* Cream surfaces */
  --cream-50:        #FBF6EE;
  --cream-100:       #F4E8D8;
  --cream-200:       #E5D8C2;

  /* Cocoa text + borders */
  --cocoa-700:       #5C4A3D;
  --cocoa-900:       #3A2E26;

  /* Accents */
  --terracotta-400:  #E59A7E;
  --terracotta-500:  #D87C5A;
  --terracotta-600:  #C26849;

  --butter-700:      #C99B30;

  --sage-500:        #88A584;
